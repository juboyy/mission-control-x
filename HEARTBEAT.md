# HEARTBEAT.md

## DNA Alignment Check (Every 2-4 Heartbeats)

**Frequency:** Rodar esta verificação a cada 2-4 heartbeats (~1-2x por dia)

**Tasks:**

### 1. Revisar Decisões Recentes vs DNA Principles
- Ler últimas entradas de `memory/YYYY-MM-DD.md` (hoje + ontem)
- Identificar decisões técnicas tomadas
- Validar contra DNA principles (Excelência, Coerência, Coesão)
- Detectar violações do Hidden OS (shortcuts, hypocrisy, disconnection)

### 2. Atualizar MEMORY.md com Lições Aprendidas
- Extrair lições significativas dos daily logs
- Sintetizar em MEMORY.md (curated long-term memory)
- Remover informações desatualizadas de MEMORY.md
- Cross-reference com DNA principles

### 3. Identificar Desvios do DNA (Reportar)
- Se detectar violação: documentar em memory/DNA-DEVIATIONS.md
- Categorizar: Intencional (justificada) vs Acidental (corrigir)
- Notificar João se desvio crítico

### 4. Sugerir Updates no DNA (Se Princípios Evoluíram)
- Novos princípios emergindo?
- Regras invisíveis mudaram?
- Assinatura criativa evoluiu?
- Se SIM: propor update para próxima DNA review (3-6 meses)
